# Layout Adjustments: Screen-by-Screen (Beginner)

Edit `LAYOUT_ADJUSTMENTS` in `js/layoutAdjustments.js`.
Numbers are treated as `px`. Use strings for units like `%`, `vw`, `vh`, or `rem`.

Property meanings (simple):
- `x`: move left or right.
- `y`: move up or down.
- `padding` and `padding*`: inner spacing around content.
- `scale`: uniform zoom for X and Y together.
- `scaleX`: width-only zoom.
- `scaleY`: height-only zoom.
- `sizeX`: width.
- `sizeY`: height.
- `positionX`: CSS `left` offset.
- `positionY`: CSS `top` offset.
- `origin`: transform pivot point.

Default reset values are shown under every element below.
